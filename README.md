# AI Stock Predictor

AI-powered stock market prediction and trading analytics platform using Python, TensorFlow, LSTM, Streamlit, Plotly, and Yahoo Finance API.

## Features
- LSTM Stock Prediction
- Technical Indicators
- Candlestick Charts
- Sentiment Analysis
- Multi-Stock Comparison
- Interactive Dashboard

## Technologies
- Python
- TensorFlow
- Streamlit
- Plotly
- Scikit-learn
- Yahoo Finance API

## Run Project

```bash
streamlit run app.py
